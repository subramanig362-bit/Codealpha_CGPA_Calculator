# CodeAlpha CGPA Calculator

This project is Task 1 of the CodeAlpha C++ Internship.  
It is a simple CGPA calculator where the user enters the grades and credits for each course, and the program calculates the final CGPA.

## Features
- Accepts number of courses
- Takes grade and credit input for each course
- Converts grade letters to grade points
- Calculates total credits
- Calculates CGPA
- Beginner-friendly C++ program
